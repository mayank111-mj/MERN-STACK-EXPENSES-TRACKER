💰 MERN Expense Tracker

This is a full-stack Expense Tracker web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). The application helps users efficiently manage their personal finances by tracking income and expenses in real-time.

🚀 Key Features:

* Add, edit, and delete transactions
* Categorize income and expenses
* Real-time balance calculation
* Secure user authentication using JWT
* RESTful API integration
* Responsive and user-friendly UI

🛠️ Tech Stack:

* Frontend: React.js (Hooks, Axios)
* Backend: Node.js, Express.js
* Database: MongoDB (Mongoose)
* Authentication: JSON Web Token (JWT)

📊 Functionality:
Users can log their daily financial transactions, view their total balance, and analyze spending habits. The system ensures secure data handling and smooth communication between frontend and backend.

🎯 Purpose:
The project demonstrates full-stack development skills, API design, authentication, and real-world problem solving using the MERN stack.

📌 Future Enhancements:

* Export reports (PDF/Excel)
* Budget planning feature
* Dark mode UI
